## Project Overview
--The purpose of this project was to showcase animation, the use of text on screen and the portrayal of storytelling with images.

--The story I was trying to tell through the webpage was the building process of a family's home. The use of typewriter and the story boxes allows the user to keep up with the "chapters" of the story. The use of the button allows the user to go through the story on their own pace.
